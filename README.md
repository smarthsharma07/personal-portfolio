# Personal Portfolio Website

A modern developer portfolio built to showcase my projects, skills, and technical journey.

🌐 **Live Website:**  
https://smarthsharma.vercel.app/

---

## About

This repository contains the source code for my personal portfolio website.

The goal of this project is to create a fast, minimal, and visually clean space where I can present:

- My projects
- Technical skills
- Development journey
- Contact information

The website is built with a modern frontend stack and deployed on **Vercel** for fast global delivery.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/smarthsharma07/portfolio-website.git
```


Navigate to the project directory:
```
cd portfolio-website
```


Install dependencies:
```
npm install
```


Run the development server:
```
npm run dev
```

Build for production:
```
npm run build
```

---

## Deployment

The portfolio is deployed using Vercel.

Every push to the main branch automatically triggers a new deployment.

---

### Customization

If you want to use this portfolio as a template:

- Fork the repository

- Replace personal information

- Update the projects section

- Deploy using Vercel or Netlify
